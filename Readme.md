## Pyomo 설명 

- Python 기반의 최적화 프로그램
  - 무료이며, 유료 및 무료 solver와 함께 사용 가능함
  - Linear Programming과 함께 Nonlinear Programming을 함께 해결할 수 있음
    - 단, Nonlinear programming은 solver의 특성을 타는 것으로 보임
  - 사이트
    - https://www.pyomo.org/
    - Ipopt solver 사용을 위하여 Conda를 이용하여 설치
